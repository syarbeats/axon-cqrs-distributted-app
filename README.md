# Axon CQRS with RabbitMQ Implementation in Spring boot
This project is a simple cqrs implementation in Springboot by using Axon Framework, 
RabbitMQ is used as message broker to handle messaging system.

## The Stacks
1. SpringBoot
2. Axon Framework
3. JGroup
4. RabbitMQ
